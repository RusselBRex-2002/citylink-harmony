# UrbanShield - Autonomous Urban Delivery Platform

UrbanShield is a mobile-first platform designed to empower citizens to monitor, report, and hold autonomous urban delivery services accountable.

## Project Overview

As autonomous drones and self-driving vehicles transform last-mile logistics, UrbanShield provides real-time visibility and safety transparency, fostering trust between delivery operators and the communities they serve.

### Key Features

- **Real-Time Visibility:** Monitor active vehicle and drone positions in your neighborhood.
- **Frictionless Reporting:** Quick, 3-tap incident reporting for safety concerns.
- **Structured Accountability:** Track case IDs with mandatory operator responses and city-level escalation.
- **Safety Dashboard:** Access zone safety scores and historical data for community awareness.
